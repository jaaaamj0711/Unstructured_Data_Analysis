
## 데이터 소개 & 분석 목적

휴대폰 센서 데이터입니다. 실험자들을 해당 기기를 장착하고 일상생활을 평소처럼 하며 데이터를 축적하였습니다. 그렇게 쌓인 데이터로 아래와 같이 5개의 행동을 분류하는 것이 목표입니다.

- 아래층 오르기
- 위층 오르기
- 앉기
- 서기
- 조깅
